# 🚀 EchoCalc AI - Quick Start Guide

## Welcome to EchoCalc!

**EchoCalc** is a voice-powered AI smart calculator with a futuristic interface that combines the elegance of Apple design with the innovation of OpenAI aesthetics.

---

## ⚡ Getting Started in 3 Steps

### Step 1: Start the Server

Open your terminal and run:

```bash
cd "ai smart calculator"
python app.py
```

The server will start at: **http://localhost:5000**

### Step 2: Open Your Browser

Navigate to: `http://localhost:5000`

You'll see the beautiful EchoCalc interface with:
- ✨ Animated gradient background
- 💎 Glassmorphism calculator panel
- 🔮 Neon glowing buttons
- 🎤 Voice input ready

### Step 3: Start Calculating!

#### Option A: Manual Input
Click buttons or type on your keyboard:
- Numbers: `0-9`
- Operators: `+`, `-`, `*`, `/`
- Press `=` or `Enter` for result

#### Option B: Voice Input (Recommended!)
1. Click the **microphone button** 🎤
2. Allow microphone access if prompted
3. Speak your calculation: *"Five plus three times two"*
4. Watch EchoCalc calculate automatically!

## ❓ Troubleshooting

### Voice Input Not Working?
1. **Check browser**: Use Chrome or Edge (recommended)
2. **Allow permissions**: Click "Allow" when asked for microphone
3. **Check mic**: Ensure microphone is connected and working
4. **Speak clearly**: Enunciate words at normal volume

### Can't See the Website?
1. **Check server**: Ensure `python app.py` is running
2. **Correct URL**: Visit `http://localhost:5000`
3. **Clear cache**: Press `Ctrl + Shift + Delete`
4. **Try different browser**: Chrome/Edge/Firefox

### Buttons Not Responding?
1. **Wait for load**: Ensure page fully loaded
2. **Check console**: Press `F12` for errors
3. **Refresh page**: Press `F5` or `Ctrl + R`
4. **Restart server**: Stop and restart Python
